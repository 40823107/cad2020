enter→輸入/y/tmp/id_rsa

下載putty、putty_home、GitExtensions

將start移到SciTE  並新增 

set GIT_HOME=%Disk%:\Portablegit\bin\
set GIT_SSH=%Disk%:\putty\plink.exe

打開.git資料夾，把config移SciTE

新增[remote "origin"]裡的url = git@github.com:40823107/cad2020.git

file→save

打開putty資料夾的puttygen.exe

點load，找到id_rsa

是→Save private key→是→40823107.ppk

打開putty.exe

若沒有github.com，在Host Name和Save Sessions打github.com按Save

設定proxy(使用自家網路、手機網路請選None)

打開SSH裡面的Auth

選擇你所生產的金鑰檔案40823107.ppk

回到session點選save再open

在home新增.ssh資料夾

ProxyCommand y:/putty/plink.exe github.com %h %p
  
Host github.com
    User git
    Port 22
    Hostname github.com
  
    TCPKeepAlive yes
    IdentitiesOnly yes

將上面的指令複製到SciTE並另存config
